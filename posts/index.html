Heaps Cool Guide Pt 1

2020-04-24
If you like making games, then the Heaps framework might be worth looking into. It is open source, works both in the browser and natively using the Hashlink virtual machine. It also has its own 3D editor, HIDE. However the documentation for the whole ecosystem might be a bit lacking, especially for some more advanced features. I have been using Heaps for a bit, and have gotten quite used to it, so I decided to write this guide for getting started with the framework, and eventually build a small game.
